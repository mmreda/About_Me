# AboutMe
The AboutMe app is a demo app that shows information about a person.
- Name
- Settable Nickname
- An image
- Scrollable information

This app demonstrates the following views and techniques:

- LinearLayout
- TextView
- EditText
- ImageView
- ScrollView
- Using data binding
- Setting multiple click handlers programmatically
